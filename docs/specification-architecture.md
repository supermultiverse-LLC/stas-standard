# STAS-01 Specification Architecture

## Purpose

This document defines the architectural structure of the STAS-01 specification.

It is not part of the normative specification.

Its purpose is to establish a stable organization for the standard before
individual technical proposals are developed through the RFC process.

The specification should evolve by expanding the sections defined here,
rather than by adding unrelated material over time.

# Evolution

Changes to the organization of the specification should be rare.

New technical functionality should normally be introduced through RFCs
that extend one of the sections defined above rather than restructuring
the specification itself.
